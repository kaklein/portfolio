Runlog
GitHub repository

An easy way for runners to log their miles, Runlog is a simple web app where users can save basic info about their running activities. The clean Runlog interface displays users' past runs using pagination and offers options to delete and edit logged activities. The app also tracks mileage totals for the current week, month, and year.
